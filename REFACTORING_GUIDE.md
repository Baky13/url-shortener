## 🔧 Настройка перед запуском

### 1. Переменные окружения
Создайте переменные окружения:

**Windows (PowerShell):**
```powershell
$env:TELEGRAM_BOT_TOKEN="ВАШ_НОВЫЙ_ТОКЕН"
$env:TELEGRAM_CHAT_ID="ВАШ_CHAT_ID"
$env:DB_PASSWORD="ВАШ_ПАРОЛЬ_ПОСТГРЕС"
```

**Windows (CMD):**
```cmd
set TELEGRAM_BOT_TOKEN=ВАШ_НОВЫЙ_ТОКЕН
set TELEGRAM_CHAT_ID=ВАШ_CHAT_ID
set DB_PASSWORD=ВАШ_ПАРОЛЬ_ПОСТГРЕС
```

**Linux/Mac:**
```bash
export TELEGRAM_BOT_TOKEN="ВАШ_НОВЫЙ_ТОКЕН"
export TELEGRAM_CHAT_ID="ВАШ_CHAT_ID"
export DB_PASSWORD="ВАШ_ПАРОЛЬ_ПОСТГРЕС"
```

### 2. Новый токен бота
**ВАЖНО**: Старый токен скомпрометирован, получите новый:

1. Зайдите в @BotFather
2. Используйте команду `/revoke` для старого бота
3. Создайте нового бота или получите новый токен
4. Обновите переменную окружения `TELEGRAM_BOT_TOKEN`

### 3. База данных
Убедитесь что PostgreSQL запущен и база `lunch_bot` создана.

Новая миграция автоматически создаст Foreign Key между таблицами.

## 🚀 Запуск приложения

```bash
# Запуск через Maven
mvn spring-boot:run

# Или через Java после сборки
mvn clean package
java -jar target/Lunch_TG_BOT-0.0.1-SNAPSHOT.jar
```
